What's inside

Five skills. One kit.

Each one solves a job I used to do badly on bad days. You get the skill file, a dedicated setup guide, and inline notes explaining every placeholder you need to fill in.

  1. 01

    Morning briefing

    A thirty-second summary every working day — what you did yesterday, what's on today, where you left off, what's on fire. Three modes: quick catch-up, full weekly view, and a mid-day handover when you switch machines. The external memory you probably wish you had.

  2. 02

    Daily log

    Capture tonight so you don't have to remember tomorrow. Smart carry-forward: deduplicates against your task list, spots when a conversation has moved on, updates aging dates when you chase someone. Feeds the morning briefing automatically.

  3. 03

    Inbox processor

    Triages your inbox into quick-replies, research, decisions, FYI, and waiting-on. Drafts replies straight to your Drafts folder (never sends). Groups similar work so you handle email in focused passes instead of one-at-a-time.

  4. 04

    Content pipeline

    A state machine for content: idea → research → outline → draft → ready → published. Pulls real sources if you have web search connected. Flags pieces stuck too long. Matches your publishing schedule so you know what's covered.

  5. 05

    Task manager

    Focus, active, recurring, backlog. A today-view that hides what doesn't matter today. Fuzzy matching finds tasks even when you shorten the name. Files, not apps — everything's a markdown file you can grep.
